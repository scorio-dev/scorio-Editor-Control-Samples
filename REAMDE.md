> A JavaScript library for detecting touch gestures.


## Installation
### NPM
```sh
npm install --save hammerjs
```

**or**

### Yarn
```sh
yarn add hammerjs
```

**or**

### CDN
[https://cdnjs.com/libraries/hammer.js/](https://cdnjs.com/libraries/hammer.js/)
